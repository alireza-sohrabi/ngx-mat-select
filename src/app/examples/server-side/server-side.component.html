Server Side
  • You need to use Server-Side mode, When there are enormous options that the loading of them costs huge then you have to load them page per page.
  • You should not use primitive values in Server-Side mode because all the options are not loaded at first and then the display-member won't be found by the Select, so you should use an object, provided with display-member and value-member as the value of the Select
  • Select always takes advantage of virtual and Infinite Scroll together in the Server-Side Mode that means don't worry about the rendered options in the DOM
